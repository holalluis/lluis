Les meves webs

[2019-2-22]

El 2011 vaig aprendre a fer webs de forma autodidacta. Vaig aprendre HTML, CSS i després Javascript. Amb el temps, el Javascript s'ha convertit en el meu llenguatge de programació preferit, junt amb el C/C++ i el Python. Mentre aprenia anava fent mini-webs molt senzilles com a exercici. Avui he estat recopilant i algunes d'aquestes webs. Algunes de les més destacades són:

  • Acudits aleatoris (la meva primera web)
  • Quants dies tens (calculadora de dies viscuts)
  • jkl (mini joc per entrenar l'agilitat mental)
  • Notes musicals (mini joc per aprendre a llegir partitures)
  • Conversor dni (converteix el teu dni a una paraula)
  • Consulta (consulta sobre la monarquia espanyola)

Aquí pots trobar la resta de webs.

Salut!
